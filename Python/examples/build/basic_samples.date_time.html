get_current_date_time(choice)¶

Displays the current time. It also calculates the related time stamp. Then converts the time stamp to date and time.

Parameters:choice (str) – The user’s selection

This function imports the datetime from the datetime module. Then, it uses the now() to get a datetime object containing current date and time. Using :py:func:datetime.strftime() method, it then creates a string representing the current time.

get_datetime_from_string(date_string)¶

Get a datetime object from a string. Example from <https://www.programiz.com/python-programming/datetime/strptime>

This function use the datetime.now().strftime() method which takes two arguments

  • string (to be converted to datetime)
  • format code

Based on the string and format code, the method returns its equivalent datetime object. For more information, see title <https://www.programiz.com/python-programming/datetime/strptime>

Note

You cannot create datetime object from every string. The string needs to be in a certain format.

param date:The string containing date and time
type date:str
return:datetime object
rtype:datetime object
